⚙️ Estimation Methodology

01

Feature Extraction

Parses the uploaded SVG floor plan using geometric scaling based on standard door widths (0.90m) to calculate GFA and count rooms.

02

Market Forecast

Uses a historical Linear Regression model (2020-2024 data) to forecast the base price/m² for the target city and completion year.

03

Cost Prediction

Applies a Linear Regression cost model incorporating room complexity: $\text{CF} = 1 + (\text{rooms} \times 0.01)$. Margin of error is given by the model's MAE (€30,882.89).
